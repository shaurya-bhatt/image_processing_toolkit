# Image Processing Toolkit

A simple Python project that demonstrates basic image processing operations such as grayscale conversion and image blurring using numerical array manipulation.

## Features

* Convert images to **grayscale**
* Apply **blur filters** to smooth images
* Understand how images are represented as **pixel arrays**

## How to Run

Clone the repository:

```
git clone https://github.com/shaurya-bhatt/image_processing_toolkit.git
```

Navigate to the project folder:

```
cd image_processing_toolkit
```

Run the main script:

```
python main.py
```

This project was created to understand how digital images can be represented as arrays and manipulated using Python.
